What is Hackathon

Hey all!!! BIAS TECHNOFORUM welcomes you all to our fresh eventâ€¦. WHAT IS HACKATHON?? A hackathon is a combination of two words hack+marathon and it is an event of any duration where people come together to solve problems. It can be students, young entrepreneurs, computer programmers and developers, graphic designers, interface designers, project managers etc.. that collaborate intensively on different projects.

IN OUR HACKATHON EVENT :- The topics we are going to covered in the event are -

  1. Women safety
  2. Environmental Awareness/Stability/Conservation
  3. Steam punks
  4. Medical issues
  5. Tours and Travels

Rules

  1. One can participate in the event as an individual or a team.
  2. The project should match the category they choose.
  3. Each/Individual should complete the prototype according to the timeline
  4. The code of the project must be uploaded to GITHUB and links should be shared in the form.
  5. Each submission must contain a short video (30-50 sec max) of the working there project.
  6. Registration will be open soon, check the timeline for the updates and register yourself.
